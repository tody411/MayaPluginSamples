Detailed Description

LaplacianSmoother implementation.

argmin_S |P - S|^2 + | L S |^2

M = Lt L S = (I + M)^{-1} P

MEL: deformer -type LaplacianSmoother; Python: cmds.deformer( type="LaplacianSmoother" );
